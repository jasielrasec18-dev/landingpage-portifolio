# Landing Page Portfólio

Este é um projeto de landing page/portfólio pessoal, desenvolvido para apresentar meus projetos, habilidades e experiências. O objetivo é fornecer uma visão clara e atraente do meu trabalho e capacidades como desenvolvedor.

## Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias e demonstra experiência com:

*   **Frontend:**
    *   HTML5
    *   CSS3 (com arquivos separados para organização: `reset.css`, `global.css`, `variables.css`, `layout.css`, `components.css`, `sections.css`)
    *   JavaScript

*   **Conhecimentos e Ferramentas Demonstradas:**
    *   React
    *   TypeScript
    *   Node.js
    *   AWS (Amazon Web Services)
    *   PostgreSQL

## Estrutura do Projeto

*   `index.html`: O arquivo principal da landing page.
*   `css/`: Contém todos os arquivos de estilo CSS, organizados para modularidade e manutenção.
*   `images/`: Armazena todas as imagens utilizadas no projeto, incluindo:
    *   `ambiente de desenvolvimento.png`, `Projeto1.png`, `Projeto2.png`, `Projeto3.png`: Imagens de projetos ou demonstrações.
    *   `icons/`: Contém ícones SVG que representam as tecnologias e ferramentas com as quais possuo experiência.

## Como Visualizar

Para visualizar este portfólio, você pode simplesmente:

1.  Clonar este repositório para sua máquina local.
2.  Abrir o arquivo `index.html` em seu navegador web preferido.

Alternativamente, você pode usar um servidor web local simples (como o Live Server para VS Code) para visualizar o projeto.
